## Attract Mode System

<table>
    <tr>
        <td colspan="1" style="text-align: center;">
        <b>
            No arcade cabinet is complete without its own attract mode, in case you’re unaware of what an attract mode is, an attract mode is a sort of “demo mode” that an arcade machine will enter when it is on the title screen and is not being actively used. While in this mode, the cabinet screen will display a demo of gameplay so that it can “attract” people walking by the cabinet to play it. I was tasked with creating the systems necessary to replicate this key part of arcade culture within Choo Choo Boom Boom, you can read the full official documentation for how I accomplished this <a href="https://docs.google.com/document/d/1yd_vJPNWCkVl43yFmMt-5p36r_LS8x82lwLE2gBODlk/edit?usp=sharing" target="_blank">here!</a>
